随机动漫图API

基本调用格式

https:///acgurl.php?return=https

参数

return=json //输出JSON格式
return=http //跳转一张http图片
return=https //跳转一张https图片
return=img //不跳转直接输出(较慢)

JSON调用格式

https:///acgurl.php?return=json

JSON数据

{
    "code":"200" #图片状态码
    "acgurl":"https:\/\/ws1.sinaimg.cn\/large\/0072Vf1pgy1foxkfy08umj31kw0w0nng.jpg" #图片地址
    "width":"2048" #图片宽
    "height":"1152" #图片高
}
